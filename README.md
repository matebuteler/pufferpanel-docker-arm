# UNOFFICIAL PufferPanel Docker images

## Images
For all architectures the following tags are available:
- `<arch>` for just the panel
- `<arch>-java` for Java 8, 16 & 17
- `<arch>-java8`
- `<arch>-java16`
- `<arch>-java17`

## Supported architectures
- arm64

## Tested on
- Ampere A1 (Oracle Cloud)
